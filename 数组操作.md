# 预分配空间

```
ans.reserve(nums.size() - k + 1);  // 预分配空间，避免push_back时重复扩容
```

只分配内存，不创建元素，`size()` 不变，必须配合 `push_back` 使用：

```cpp
v.reserve(5);
v.push_back(1);  // OK
v[0] = 1;        // UB！元素不存在
```

# 预创建元素（resize）

```cpp
v.resize(5);  // 分配内存并创建5个元素（默认值0），size=5
v[0] = 1;     // OK，元素存在
```

需要用下标 `v[i]` 赋值时用 `resize`，只用 `push_back` 时用 `reserve`。
