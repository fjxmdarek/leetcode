# 234. 回文链表 (Palindrome Linked List)

### 题目描述

给你一个单链表的头节点 `head` ，请你判断该链表是否为回文链表。如果是，返回 `true` ；否则，返回 `false` 。

### 示例

**示例 1：**

> **输入：** head = [1,2,2,1]
> **输出：** true

**示例 2：**

> **输入：** head = [1,2]
> **输出：** false

### 提示

* 链表中节点数目在范围 `[1, 10^5]` 内
* `0 <= Node.val <= 9`

### 进阶

* 你能否用 $O(n)$ 时间复杂度和 $O(1)$ 空间复杂度解决此题？

---

### 解题思路（快慢指针 + 反转后半部分）

如果使用数组存储节点值再进行双指针判断，空间复杂度为 $O(n)$。为了达到 $O(1)$ 的进阶要求，我们需要直接在原链表上操作：

1. **寻找中点**：

   - 使用 **快慢指针 (Slow & Fast pointers)**。快指针走两步，慢指针走一步。
   - 当快指针到达末尾时，慢指针恰好指向链表的中间（或前半部分的末尾）。
2. **反转后半部分**：

   - 从慢指针的下一个节点开始，调用“反转链表”的逻辑，将后半部分完全翻转。
3. **回文比较**：

   - 比较前半部分（从 `head` 开始）和反转后的后半部分。
   - 如果所有值都相等，则为回文。
4. **还原链表（可选，但推荐）**：

   - 为了不破坏原始结构，比较完成后可以将后半部分再次反转还原。
