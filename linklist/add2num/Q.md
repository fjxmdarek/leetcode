# 2. 两数相加 (Add Two Numbers)

### 题目描述

给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。
你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

### 示例

**示例 1：**

> **输入：** l1 = [2,4,3], l2 = [5,6,4]
> **输出：** [7,0,8]
> **解释：** 342 + 465 = 807.

**示例 2：**

> **输入：** l1 = [0], l2 = [0]
> **输出：** [0]

**示例 3：**

> **输入：** l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
> **输出：** [8,9,9,9,0,0,0,1]

### 提示

* 每个链表中的节点数在范围 `[1, 100]` 内
* `0 <= Node.val <= 9`
* 题目数据保证列表表示的数字不含前导零

---

### 解题思路（模拟竖式加法）

由于链表是 **逆序** 存储的，这意味着链表的头节点就是 **个位**。这非常方便我们直接从头开始遍历并相加。

1. **初始化**：

   - 创建一个 **哨兵节点 (Dummy Node)** `prehead`，方便返回结果。
   - 使用指针 `curr` 指向当前正在构建的结果链表节点。
   - 定义变量 `carry`（进位），初始为 0。
2. **迭代遍历**：

   - 只要 `l1` 不为空、`l2` 不为空，或者 `carry` 不为 0，就继续循环。
   - 获取当前位的值：`x = l1.val`（若 `l1` 已空则为 0），`y = l2.val`（若 `l2` 已空则为 0）。
   - 计算和：`sum = x + y + carry`。
   - 更新进位：`carry = sum / 10`。
   - 创建新节点：其值为 `sum % 10`，接在 `curr.next` 上。
   - 移动指针：`l1`、`l2`、`curr` 分别向后移动。
3. **结束返回**：

   - 返回 `prehead.next`。

---

### 复杂度分析

* **时间复杂度**：$O(\max(m, n))$。其中 $m$ 和 $n$ 为两个链表的长度，我们需要遍历较长的那个。
* **空间复杂度**：$O(1)$。注意：返回值链表不计入额外空间。我们只使用了常数个变量。
