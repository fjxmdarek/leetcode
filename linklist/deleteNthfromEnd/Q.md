# 19. 删除链表的倒数第 N 个结点 (Remove Nth Node From End of List)

### 题目描述

给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

### 示例

**示例 1：**

> **输入：** head = [1,2,3,4,5], n = 2
> **输出：** [1,2,3,5]

**示例 2：**

> **输入：** head = [1], n = 1
> **输出：** []

**示例 3：**

> **输入：** head = [1,2], n = 1
> **输出：** [1]

### 提示

* 链表中结点的数目为 `sz`
* `1 <= sz <= 30`
* `0 <= Node.val <= 100`
* `1 <= n <= sz`

### 解题思路（快慢指针 / 先行指针）

为了在一次遍历中找到倒数第 $N$ 个节点，我们可以利用两个指针之间的“间距”。

1. **引入哨兵节点 (Dummy Node)**：

   - 创建 `dummy` 指向 `head`。这主要是为了处理删除“头节点”的情况（例如链表长度为 2，删除倒数第 2 个）。
2. **初始化双指针**：

   - `fast` 和 `slow` 初始都指向 `dummy`。
3. **拉开间距**：

   - 让 `fast` 先向前移动 `n + 1` 步。
   - 此时，`fast` 和 `slow` 之间隔了 `n` 个节点。
4. **同步移动**：

   - 同时移动 `fast` 和 `slow`，直到 `fast` 指向 `null`（即到达链表末尾的后一个位置）。
   - 此时，`slow` 恰好指向 **倒数第 $N+1$ 个节点**（即要删除节点的前驱节点）。
5. **执行删除**：

   - `slow.next = slow.next.next`。
   - 返回 `dummy.next`。
