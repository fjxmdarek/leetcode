# 438. 找到字符串中所有字母异位词 (Find All Anagrams in a String)

### 题目描述

给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **字母异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

**字母异位词** 指字母相同，但排列不同的字符串。

### 示例

**示例 1：**

> **输入：** s = "cbaebabacd", p = "abc"
> **输出：** [0,6]
> **解释：**
> 起始索引等于 0 的子串是 "cba", 它是 "abc" 的字母异位词。
> 起始索引等于 6 的子串是 "bac", 它是 "abc" 的字母异位词。

**示例 2：**

> **输入：** s = "abab", p = "ab"
> **输出：** [0,1,2]
> **解释：**
> 起始索引等于 0 的子串是 "ab", 它是 "ab" 的字母异位词。
> 起始索引等于 1 的子串是 "ba", 它是 "ab" 的字母异位词。
> 起始索引等于 2 的子串是 "ab", 它是 "ab" 的字母异位词。

### 提示

* `1 <= s.length, p.length <= 3 * 10^4`
* `s` 和 `p` 仅包含小写字母

---

### 解题思路（定长滑动窗口 + 词频数组）

由于字母异位词的长度必须等于 `p` 的长度，我们可以维护一个长度为 `len(p)` 的 **固定窗口**。

1. **统计词频**：
   - 使用两个长度为 26 的数组（或哈希表）来分别记录字符串 `p` 的词频和当前窗口 `s[left...right]` 的词频。
2. **初始化窗口**：
   - 先统计 `s` 中前 `len(p)` 个字符的词频。
   - 如果初始窗口的词频数组与 `p` 的相等，记录索引 `0`。
3. **滑动窗口**：
   - 窗口每次向右移动一位：
     - **右进**：加入一个新的字符 `s[i]`，更新窗口词频。
     - **左出**：移除窗口左端的一个字符 `s[i - len(p)]`，更新窗口词频。
   - 每次移动后，比较两个词频数组是否相同。
4. **比较技巧**：
   - 在 Python 或 Java 中，可以直接比较两个数组/集合是否相等。

---

### 时间复杂度

$O(n + m)$，其中 $n$ 是 $s$ 的长度，$m$ 是 $p$ 的长度。虽然每次移动都要比较数组（复杂度 26），但 26 是常数，所以整体仍为线性时间。
